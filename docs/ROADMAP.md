# 路线图（ROADMAP）

更新时间：2026-02-18

## 1. 里程碑状态

## M1（已完成）
- 聊天主链路
- Notion 保存
- 设置中心基础能力

## M2（已完成）
- SQLite 持久化
- 多会话管理
- 页面与交互重构（含设置弹窗）

## M3（已完成）
- Agent Tool Calling 循环
- 联网搜索工具接入（多 provider）
- 来源附带与前端展示

## M4（已完成）
- 本地接入与权限（白名单）
- 本地文件读取（txt/pdf/office）
- 本地文件操作（创建/写入/复制/移动/重命名/删除）
- 按意图自动保存到 Notion（支持命名）

## M5（进行中）
- 稳定性与治理
- 工具超时、重试、降级策略
- 高风险操作策略开关与更细粒度权限
- 文档与测试体系完善

## 2. 当前优先级（建议）

1. 稳定性
- tool timeout/retry 统一策略
- 搜索 provider 回退可观测日志

2. 安全治理
- 删除/覆盖/移动的策略开关（设置页）
- 更明确的危险操作确认协议

3. 体验优化
- 工具过程展示精简模式（可折叠/可开关）
- 失败提示更可执行（给出下一步建议）

## 3. 中期规划（M6）

- 本地文件分块 + 长文 map-reduce 总结
- 工具并行与计划执行（多步任务更稳）
- 本地知识索引与检索增强
- 官方 Notion OAuth 与更完整官方能力接入

## 4. 延后项

- Edge 收藏自动入库完整产品线（采集、抽取、队列、去重）
- 企业级权限、多用户隔离
